# 🏋️‍♂️ FitZone v9

Aplicación desarrollada para la gestión del sistema FitZone. Backend construido con Spring Boot y base de datos **MongoDB**.

---

## 📌 Requisitos

Antes de ejecutar el proyecto, asegúrate de tener:

* ☕ Java JDK 17 o superior
* 📦 Maven (o usar el wrapper incluido)
* 🍃 MongoDB instalado y en ejecución
* 🖥️ Git
* 🧩 Visual Studio Code

---

## 📥 Clonar el repositorio

```
git clone https://github.com/jjcostaa/Proyecto-Aula.git
cd Proyecto-Aula
```

---

## 🗄️ Configuración de la base de datos (MongoDB)

1. Asegúrate de que MongoDB esté ejecutándose en tu equipo

En Windows puedes iniciarlo con:

```powershell 
net start MongoDB
```



## 🚀 Ejecutar el proyecto en Visual Studio Code

1. Abre Visual Studio Code
2. Selecciona **File → Open Folder** y abre la carpeta del proyecto
3. Espera a que VS Code cargue las dependencias automáticamente
4. Asegúrate de tener instalada la extensión **Java Extension Pack**
5. Ve al archivo principal del proyecto (clase con `@SpringBootApplication`)
6. Haz clic en **Run** o presiona **Ctrl + F5**

---

## 🌐 Acceso

Una vez iniciado el servidor:

http://localhost:8080
